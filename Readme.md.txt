# 📋 Sistem Informasi Planning Produksi

Sistem Informasi Planning Produksi merupakan aplikasi berbasis web yang dikembangkan untuk membantu proses perencanaan, pengendalian, dan monitoring produksi pada perusahaan manufaktur. Sistem ini mendukung pengelolaan Sales Order, proses Planning Produksi, monitoring Finishing, serta manajemen pengguna dalam satu platform yang terintegrasi.

Aplikasi ini dikembangkan sebagai proyek Tugas Akhir (Skripsi) sekaligus menjadi portofolio pengembangan aplikasi berbasis PHP Native, MySQL, dan Automation Testing menggunakan Cypress.

---

## ✨ Fitur

- Login & Authentication
- Dashboard Monitoring
- Manajemen User
- Manajemen Sales Order
- Planning Produksi
- Monitoring Finishing
- Monitoring Progress Produksi
- Automation Testing menggunakan Cypress

## ⚙️ Cara Menjalankan

### 1. Clone Repository

```bash
git clone https://github.com/USERNAME/sistem-informasi-planning-produksi.git
```

> Ganti `USERNAME` dengan username GitHub Anda.

### 2. Letakkan Project

Simpan project pada folder:

```
xampp/htdocs/
```

### 3. Import Database

Import file database (`.sql`) menggunakan phpMyAdmin.

### 4. Jalankan Server

Aktifkan:

- Apache
- MySQL

menggunakan XAMPP.

### 5. Akses Aplikasi

```
http://localhost/Sistem-Informasi-Planning-Produksi
```

---

## 🧪 Automation Testing

Install dependency:

```bash
npm install
```

Menjalankan Cypress:

```bash
npx cypress open
```

atau

```bash
npx cypress run
```
